Demo Purpose

The purpose of this demo is to show that we can rapidly build websites, while solving legitimate business needs & also allowing for site customization.

This site showcases that:

  1. We can easily roll out websites on a fast-paced schedule.
  2. We can seemlessly mix external 3rd-party, 4th-party, 5th-party, 6th-party, 7th-party, ... nth-party library technologies together, to create radical websites. We’re simply showcasing that we can mix libraries, without any library conflicts. So we can select from any of these & use them as needed on future projects:
    • Angular JS
    • Bootstrap JS
    • D3JS
    • FamFamFam icons
    • Jasmine
    • jQuery
    • jQuery Mobile
    • jQuery QueryBuilder plug-in
    • jQuery UI
    • Moment JS
    • Underscore.js
  3. We can integrate Unit Tests into our JavaScript code, as needed.
  4. We can internationalize websites, into any language that the customer desires.
  5. We can build data-driven websites.
  6. We can re-skin them once we are finished building them.
  7. We can build wonderful websites!

We&re not showcasing that we are using all of those items for this demo, but we are showing that we can use them as needed on our next project. We have the experience to build amazing looking websites, which use these libraries.
